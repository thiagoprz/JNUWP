# Aplicativo UWP (Universal Windows Platform) não oficial do Jovem Nerd
Como o 2% dos usuários de smartphone não foram privilegiados com um aplicativo do Jovem Nerd decide criar este app para suprir algumas funcionalidades extras que o podcast tem e que não ficam disponíveis em apps de podcast comúns, como as insertions, que são inserções de imagens de assuntos ou coisas mencionadas durante a reprodução.

## Instruções
Baixe um cópia para seu local e abra o projeto JovemNerd.sln no Microsoft Visual Studio 2015.

### Prerequisities
Sistema Operacional: Windows 10 (de preferência com a atualização de aniversário do Windows) pode ser pc ou mobile.
Visual Studio 2015 Update 3

### Instalação
A instalação é simples bastando realizar o deploy através do Visual Studio.
